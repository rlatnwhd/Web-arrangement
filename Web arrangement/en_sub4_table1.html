<table> / <tr>

<table>태그는 표 자리를 만드는 태그, <tr>태그는 표의 행(가로줄) 영역
 - 표 생성시 반드시 <table>태그를 사용하여 안에 만들어야 함
- <tr>태그 하나당 하나의 행을 생성함
- <ul> / <ol>태그와 사용법이 같음

<td> / <th>

<td>태그는 표의 셀(항목)을 만드는 태그, <th>태그는 제목 셀을 만드는 태그
 - <td>태그 하나당 한 개의 셀이 생성됨
- 반드시 <tr>태그 안에 생성
- <li>태그와 사용법이 같음
- colspan, rowspan 속성을 사용하여 셀을 합칠 수 있음
속성 설명
colspan colspan="(해당 셀 포함 개수)" 가로줄의 셀을 합칩니다. 해당 셀을 기준으로 오른쪽 셀이 합쳐집니다.
rowspan rowspan="(해당 셀 포함 개수)" 세로줄의 셀을 합칩니다. 해당 셀을 기준으로 아래쪽 셀이 합쳐집니다.
- colspan, rowspan 속성을 사용하면 합쳐진 공간의 셀은 없어도 됨

<caption>

<caption>태그는 표에 제목을 붙히는 태그
 - <caption>태그를 사용하면 표 위에 제목 영역이 생김

<figcaption>

<figcaption>태그는 표에 제목을 붙히는 태그
 - <caption>태그와 똑같이 표 위에 제목 영역이 생김
- <caption>태그와 다른 점은 글이 중앙에 정렬되지 않음
- <figure>태그 안에 생성

<thead>

<thead>태그는 표의 제목 부분을 정의하는 태그
 - 표의 맨 위의 행을 <thead>태그로 제목 부분을 정의할 수 있음
