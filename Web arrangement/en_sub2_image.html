<img>

<img>태그는 이미지를 삽입하는 태그
 - <img>태그는 scr="(경로)" 라는 경로를 지정하는 속성과 함께 쓰임
- 이미지가 같은 경로에 있다면 이미지 이름만, 다른 하위 폴더에 들어있다면 (하위 폴더 이름)/(이미지 이름) 을 사용
- 폴더의 이미지도 연결할 수 있지만, 사이트의 이미지도 (사이트 url 입력) 연결
- alt="(내용)" 속성은 이미지가 존재하지 않아 표시가 되지 않을 때 사진 대신 보여주는 대체 텍스트
- 이중 태그

<figure>

<figure>태그는 설명 글을 붙힐 대상을 지정하는 태그
 - 웹 문서에서 오디오나 비디오 같은 멀티미디어 파일을 비롯해 사진이나 표, 소스 코드 등 웹 문서 안에서 한 단위가 되는 요소를 묶음
- 이미지에 설명 글을 넣기 위해선 <figure>태그로 묶어야 함
- 이중 태그

<figcaption>

<figcaption>태그는 설명 글을 붙히는 태그
 - 붙일 수 있는 대상은 텍스트뿐만 아니라 이미지나 오디오, 비디오 같은 미디어 파일이나 표가 될 수도 있음
- 여러 개의 이미지나 미디어 파일 하나에 설명 글 표시 가능
- 이미지가 <figure>태그 안에 있기 때문에 이미지를 옮기면 <figcaption>태그도 같이 옮겨짐

<a>

<a>태그는 하이퍼링크를 만드는 태그
 - <a>태그의 속성 종류는 다음과 같음 ( <a (속성)=""></a> )
속성 설명
href 링크한 문서나 사이트의 주소를 입력합니다.
target 링크한 내용이 표시될 위치(현재 창 또는 새 창)를 지정합니다.
download 링크한 내용을 보여 주는 것이 아니라 다운로드 합니다.
rel 현재 문서와 링크한 문서의 관계를 알려줍니다.
hreflang 링크한 문서의 언어를 지정합니다.
type 링크한 문서의 파일 유형을 알려줍니다.
- target의 속성 값은 다음과 같음 ( <a target="(속성 값)"></a> )
속성 값 설명
_blank 링크 내용이 새 창이나 새 탭에서 열립니다.
_self target 속성의 기본 값으로 링크가 있는 화면에서 열립니다.
_parent 프레임을 사용했을 때 링크 내용을 부모 프레임에 표시합니다.
_top 프레임을 사용했을 때 프레임에서 벗어나 링크 내용을 전체 화면에 표시합니다.
- 긴 페이지 화면일 때 앵커(class, id)를 사용하여 해당 앵커의 요소로 화면이 이동됨
- SVG 이미지를 사용하면 이미지를 확대, 축소했을 때 이미지의 깨짐 같은 현상이 일어나지 않음
