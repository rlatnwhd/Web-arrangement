margin

margin
 - margin= <숫자> 로 바깥 여백을 설정 가능
- margin의 <숫자> 를 하나만 사용 시 상하좌우 전부 바깥 여백 적용
- margin= <숫자> <숫자> 로 사용 시 상하 -> 좌우 순서로 적용됨
- margin= <숫자> <숫자> <숫자> <숫자> 로 사용 시 상 -> 우 -> 하 -> 좌 순으로 적용됨
- margin-(상하좌우)= <숫자> 로 지정 시 해당 부분만 바깥 여백 적용

overflow

overflow
 - hidden으로 설정 시 해당 구역을 벗어나는 인근 요소들은 숨겨버림

padding

padding
 - padding= <숫자> 로 안쪽 여백을 설정 가능
- padding의 <숫자> 를 하나만 사용 시 상하좌우 전부 안쪽 여백 적용
- padding= <숫자> <숫자> 로 사용 시 상하 -> 좌우 순서로 적용됨
- padding= <숫자> <숫자> <숫자> <숫자> 로 사용 시 상 -> 우 -> 하 -> 좌 순으로 적용됨
- padding-(상하좌우)= <숫자> 로 지정 시 해당 부분만 안쪽 여백 적용

text

text-align
 - 텍스트를 정렬할 수 있음(왼쪽, 가운데 오른쪽 정렬)

text-decoration
 - 링크의 밑줄 같은 요소를 추가하거나 제거할 수 있음

z-index

z-index
 - 구역의 우선순위를 설정할 수 있음
- 숫자가 높을수록 앞에 옴(음수도 가능(뒤로 감))

list

list-style
 - 리스트 앞에 붙는 요소를 추가하거나 제거할수 있음(모양 바꾸기도 가능)
