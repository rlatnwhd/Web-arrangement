HTML 태그로 기본적인 골격은 모두 갖추었는데 웹 문서가 심심하게 느 껴지나요? 그림이나 글자들이 정리되지 않은 채 나열되어
어설퍼 보이 나요? 지금부터 이 문서에 옷을 입혀 보겠습니다. HTML로 만들었던 골격에 디자인으로 옷을 입혀 주는 것이 바로 CSS(Cascading Style Sheets)입니다. 웹 디자인이나 웹 개발에서 CSS는 빼놓고 말할 수 없을 정도로 중요합니다. HTML보다 더 많은
역할을 한다고 할 수 있죠.

background

background
 - 색깔과 url()을 사용하여 이미지를 지정할 수 있음

background-color
 - 색깔을 지정할 수 있음

background-image
 - url()을 사용하여 이미지를 지정할 수 있음

background-position
 - 배경화면의 요소의 위치를 지정할 수 있음

background-size
 - 배경화면 요소의 크기를 지정할 수 있음

background-repeat
 - 배경화면의 요소의 반복을 지정할 수 있음

border

border
 - ( border <두께> <선 종류> <색깔>) 형식으로 테두리를 지정할 수 있음
- border- 다음에 위치(상하좌우)를 지정하여 그 부분만 테두리를 지정할 수 있음

border-style
 - 테두리 선의 종류를 지정할 수 있음

border-color
 - 테두리 선의 색깔을 지정할 수 있음

border-radius
 - 테두리의 모서리를 둥글게 지정할 수 있음
- border-(상하좌우)-(상하좌우)를 지정하여 그 부분만 둥근 모서리를 지정할 수 있음

box-sizing

box-sizing
 - content-box - width의 속성 값을 콘텐츠 영역 값으로 사용
- border-box - width의 속성 값을 콘텐츠 영역에 테두리까지 포함한 박스 전체 너비 값으로 사용

shadow

box-shadow
 - ( box-shadow: <수평 거리> <수직 거리(X,Y)> <흐림 정도> <번짐 정도> <그림자 색깔> ; ) 형식으로 구역 그림자를 만들수 있음(안족 그림자는 inset)

text-shadow
 - box-shwdow와 사용법이 같음

color

color
 - 해당 구역에 글자색을 설정할 수 있음

font

font-size
 - 글자의 크기를 설정할 수 있음

font-weight
 - 글자의 굵기를 설정할 수 있음

font-family
 - 글자의 폰트를 설정할 수 있음
- 쉽표(,)로 구분하고 나열하여 여러 개 설정 가능(왼쪽 부터 순서대로)(적용이 안될 시 오른쪽으로 한 칸 씩 이동)

float

float
 - 구역을 공중으로 띄우고 위치 속성 값(left, right)을 사용하여 해당 위치로 정렬할 수 있음
- 인접한 아래쪽 형제 요소도 float 영향을 받으므로 해당 형제 요소에 clear= both; 를 주어야 함
- 공중에 띄워져 있기 때문에 요소가 겹쳐질 수 있음

line

line-height
 - 줄 간격을 설정할 수 있음
