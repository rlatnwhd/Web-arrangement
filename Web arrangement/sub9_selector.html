선택자(selector)를 속성의 맨 앞에 붙여 속성이 어디에 적용되는지 표시했습니다. 앞에서 배웠던 주요 선택자 외에도 CSS3에는
매우 다양한 선택자들이 있습니다. 이런 선택자를 사용하면 사용자 동작에 반응하는 웹 문서뿐만 아니라 애니메이션까지
만들 수 있죠.

CSS3 선택자

연결 선택자 속성 선택자 가상 클래스 가상 요소

연결 선택자

하위 선택자
 - 지정한 모든 하위 요소에 스타일을 적용함
- 정의할 때는 상위 요소와 하위 요소를 공백으로 구분하고 대괄호 안에 속성을 지정
- 자식 요소뿐만 아니라 손자 요소, 손자의 손자 요소 등 모든 하위 요소까지 적용됨

자식 선택자
 - 자식 요소에만 스타일을 적용됨
- 하위 요소 전체에 스타일을 적용하는 것이 아니라 자식 요소에만 스타일 지정 가능
- 정의할 때는 두 요소 사이에 부등호를 표시해 부모 요소와 자식 요소를 구분
- 손자 요소까지 스타일이 적용되는게 아닌 바로 아래 요소인 자식 요소만 스타일이 적용됨

인접 형제 선택자
 - 가장 가까운 형제 요소에 스타일을 적용
- 정의할 때는 상위 요소와 하위 요소를 "+" 로 묶음
- 가까운 요소만 정의 가능

형제 선택자
 - 형제 요소에 스타일을 적용
- 정의할 때는 상위 요소와 하위 요소를 "~" 로 묶음

속성 선택자

- 속성 선택자의 종류는 다음과 같음
기본형 설명
[속성] 지정한 "속성"일 때
[속성 = "값"] "값"이 일치할 때
[속성 ~= "값"] 여러 값들 중 특정 "값"이 포함되어 있을 경우(단어별)
[속성 != "값"] "값"이 포함되어 있을 때(하이픈 포함 단어별)
[속성 ^= "값"] "값"으로 시작할 때
[속성 $= "값"] "값"으로 끝날 때
[속성 *= "값"] "값"이 속성 값이 일부일 때

가상 클래스

:link
 - 문서 안의 하이퍼링크 중에서 사용자가 아직 방문하지 않은 링크에 스타일을 적용함
- 기본은 파란색과 밑줄

:visited
 - 문서 안의 하이퍼링크 중에서 한 번 이상 방문한 링크에 대한 스타일을 적용함
- 기본은 자주색과 밑줄

:hover
 - 웹 요소 위로 마우스 커서를 올려놓을 때의 스타일을 지정함

:action
 - 링크나 이미지 등 웹 요소를 활성화했을 때(누르고있을 때)의 스타일을 지정함

:focus
 - 웹 요소에 초점이 맞추어졌을 때의 스타일을 지정함

:enabled / :disabled
 - 요소를 사용할 수 있을 때와 없을 때의 스타일을 지정함

:checked
 - 라디오 박스나 체크 박스에서 해당 항목을 선택했을 때의 스타일을 지정함

:root
 - 문서 전체에 스타일을 적용함

:nth-child(n) / :nth-last-child(n)
 - 자식 요소의 위치에 따라 스타일을 적용함

:nth-of-type(n) / :nth-last-of-child(n)
 - 특정 태그 위치에 스타일을 적용함

:first-child / :last-child
 - 첫번째, 마지막 요소에 스타일을 적용함

:first-of-type / :last-of-type
 - 형제 관계 요소의 위치에 따라 스타일을 적용함

:only-child / :only-of-type
 - 하나뿐인 자식 요소에 스타일을 적용함

:target
 - 앵커 목적지에 스타일을 지정함

:not
 - 특정 요소가 아닐 때 스타일을 적용함

가상 요소

::first-line / ::first-letter
 - 첫번 째 줄, 첫 번째 글자에 스타일을 적용함

::before / ::after
 - 내용 앞뒤에 콘텐츠를 추가함
